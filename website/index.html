Frequently asked questions

    
What you need to know.

    

      

        What is Axiom?
        
A Roblox script loader. You paste one line into your executor; it loads the right script from GitHub (MM2 or universal) and runs it.

      

      

        Which games are supported?
        
Murder Mystery 2 (dedicated build) and a universal build for other games. The loader picks the right one by game PlaceId.

      

      

        How do I use it?
        
Copy the loader code from this page, paste it into your Roblox executor, and execute. The script UI will load in-game.

      

      

        What executors work?
        
Any executor that supports loadstring and HttpGet (e.g. Delta, Synapse, Wave). Paste the loader and run.
